# Personal Finance Tracker
A Java-based desktop application that allows users to track income and expenses, as well as analyze spending habits.

## Features
- Add, edit, and delete income and expense transactions
- Categorize transactions for better analysis
- Desktop GUI built with Java Swing
- Input validation and error handling

## Tech Stack
- Language: Java
- GUI: Java Swing
- Data Handling CSV (file import/export)
- IDE: Eclipse

## Motivation
I built this project to strengthen my understanding of object-oriented design, database integration with JDBC, and building desktop applications with a graphical user interface. It also helped me practice managing real-world data and designing a project end-to-end.
